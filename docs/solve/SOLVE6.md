### 🚀 기능 목록

1. [예외처리] 이메일,닉네임 입력 
    - 이메일
      - 도메인은 `email.com`만 가능
      - 길이는 11자 이상 20자 미만
    - 닉네임
      - 완전한 한글만 가능 (모음, 자음 불가능)
      - 길이는 1자 이상 20자 미만 
2. [요구] 같은 글자가 연속적으로 포함 되는 닉네임을 작성한 지원자의 이메일 목록 생성
3. [요구] 이메일에 해당하는 부분의 문자열을 오름차순으로 정렬하고 중복은 제거
4. 결과 출력
    - 이메일 목록
    - 예외사항: null
